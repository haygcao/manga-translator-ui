# v2.1.3 更新日志

发布日期：2026-02-27

## 🐛 修复

- 修复智能缩放/智能气泡模式下，竖排 `has_br=False` 场景断句失效：竖排断句预处理顺序调整为先规范化 `[BR]/<br>/【BR】` 再注入 `<H>` 标签，避免换行标记被标签化后失效。
- 修复模型辅助合并在 `other` 包裹场景下的漏合并：特殊合并阶段允许 `other` 作为包裹关系桥接参与分组；`other` 仍仅作辅助，不参与最终文本块几何合并结果。
- 回退 inpainting 画布级额外 `padding + 裁回` 逻辑，仅保留原有 ONNX/模型内部对齐路径，避免引入额外尺寸处理分支。
- Desktop UI：翻译任务完成后增加 `torch.cuda.empty_cache()` 调用，并将该提示日志级别调整为 `DEBUG`，减少 INFO 日志噪音。
- 修复 `load_text` 导入 JSON 蒙版时的尺寸错配：进入 inpainting 前统一将 `mask_raw/mask` 对齐到当前 `img_rgb` 尺寸并规整为 `uint8`，避免 ONNX `Mul` 广播维度错误（如 `1408x2758` 不一致）。
